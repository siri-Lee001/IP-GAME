# 角色多视角定妆图提示词模板（不限物种）

用途：
- 在节点场景图、故事板图、结局图之前，先生成“角色定妆/多视角角色图”
- 作为后续所有生图与视频参考图的身份锚点
- 适用于：人类、动物、妖怪、精灵、机器人、拟人化生物、神兽等

变量版提示词（将 `{{...}}` 替换为具体角色信息）：

```text
21:9超宽横版角色完整设定卡，纯白干净棚拍背景。

以“{{CHARACTER_NAME}}”为唯一身份锚点：必须严格保持以下身份特征一致：
- 物种/形态：{{CHARACTER_KIND}}
- 年龄感/气质：{{AGE_TEMPERAMENT}}
- 面部或头部轮廓：{{HEAD_FACE_STRUCTURE}}
- 眼部特征：{{EYE_FEATURES}}
- 鼻口/吻部特征：{{NOSE_MOUTH_FEATURES}}
- 发际线、发型结构、毛发/羽毛/耳朵/角/触角/头饰等顶部特征：{{TOP_FEATURES}}
- 服装、配饰、鞋履或肢体末端装备：{{COSTUME_ACCESSORIES}}
- 关键识别点：{{IDENTITY_MARKERS}}

只允许同一个角色，禁止换脸、禁止五官或头部结构漂移、禁止发型/毛发结构简化、禁止头饰缺失、禁止物种特征丢失。
如果角色是动物、神兽、妖怪、机器人或其他非人角色，必须严格保留其原始种属结构与比例，不得擅自人类化。

单张合成图，左中右三分区构图，三区统一光影与色彩，柔光棚拍布光，光源方向一致。

左区（占画面宽度约25%）：人物面部正面超高清特写（头到胸/上半身），头顶关键结构完整入画不裁切。
中区（占画面宽度约45%）：三张全身站姿并排（正/侧/背），头顶与脚底对齐，角色高度一致。
右区（占画面宽度约30%）：2列×3行肖像网格（不同角度/轻微表情），但仍保持同一脸同一发型结构。

质感与画质：{{VISUAL_STYLE}}；高端写实棚拍/电影级角色设定质感；8K细节；轻胶片颗粒；超干净白底。

强约束：
- 画面内不允许任何可读文字（不要 FRONT/SIDE/BACK 标签、不要角度标注、不要表情标注）
- 不要字幕、不要logo、不要UI叠层、不要水印块
- 不要多余人物/多余生物
- 不要畸形肢体/多肢体/脸崩/眼神涣散/关键器官缺失
- 全图所有分区必须是同一角色（同一脸同一发际线同一发型/毛发结构/物种结构）
```
